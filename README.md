# Malaika Portfolio

Job-ready Full Stack Developer portfolio built with **Next.js 14** (App Router), **Tailwind CSS**, and **TypeScript**.

## Features

- Dark theme with brand colors (Primary, Secondary, Accent)
- Sticky navbar with smooth scroll
- Hero, About, Skills, Projects, Contact, Footer sections
- Responsive layout and subtle scroll animations
- SEO-friendly metadata
- Accessible forms and buttons

## Setup

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## Customize

- **Resume**: Add your `resume.pdf` to the `/public` folder so the "Download Resume" button works.
- **Contact links**: Update Email, LinkedIn, and GitHub URLs in `components/Contact.tsx`.
- **Project links**: Set Live Demo and GitHub URLs in `components/Projects.tsx`.
- **Project image**: Replace the placeholder in `components/Projects.tsx` with an image (e.g. using `next/image`).

## Tech Stack

- Next.js 14 (App Router)
- Tailwind CSS
- TypeScript
